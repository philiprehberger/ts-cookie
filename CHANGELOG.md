## 0.1.0

- Initial release
- `parseCookies()` for parsing Cookie headers
- `parseCookie()` for parsing Set-Cookie headers
- `serializeCookie()` for building Set-Cookie strings
- `cookieBuilder()` with chainable API
- `signCookie()` and `verifyCookie()` for HMAC-SHA256 signing
- `getCookie()`, `setCookie()`, `deleteCookie()` for browser client
- Full TypeScript types for all cookie attributes
